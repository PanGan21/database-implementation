# database-implementation

## Simplified database impelementation of sql lite architecture
[Sql Lite Architecture](https://www.sqlite.org/arch.html)
</br>
Includes:
- Input buffer
- Table abstraction
- Cursor abstraction
- Pager abstraction
- B trees
- B trees binary search and insert
- Persist data to disk
- Query parser

#### Run the database
```make clean```
</br>
```make db```
</br>
```./db data.db```
